# mealwise

## サービス概要
mealwise は、**食費を予算内に抑えること**を最優先にした、個人向けの献立・買い物・価格・食事記録Webサービスです。日常利用を前提に、スマホから短時間で入力・確認できる運用を目指します。

## 解決したい課題
- 月末に食費が想定より増える
- 献立と買い物が分断され、無駄買いが発生する
- 外食・コンビニ・飲み物・お菓子を含めた実支出が見えにくい
- 実際の購入価格が蓄積されず、翌月の計画精度が上がらない

## 初期ターゲット
- 一人暮らし
- 夫婦
- 同棲世帯

## 主な機能（計画）
- 予算管理（月予算・週予算）
- 食事対象ルール（曜日、食事枠、特定日例外）
- レシピ管理（保存・お気に入り・タグ/フォルダ）
- ルールベース中心の献立提案
- 買い物リスト生成（食材自動合算、購入済みチェック）
- 購入価格記録と平均価格、今月予想価格
- 食事実績の簡易記録
- AI補助（将来段階で導入）

## MVP方針
- **MVP 0**: 日常利用できる最小版（認証・データ分離を含む）
- **MVP 1**: 実用性強化（AI補助、栄養表示、通知など）
- **MVP 2**: 実ユーザー運用強化（家族共有、連携、管理機能強化）

詳細は `docs/requirements.md` と `docs/roadmap.md` を参照。

## 開発運用方針
- ドキュメント・PR・レビューは日本語で運用
- 予算計算/価格計算などの重要ロジックはUI層に直接書かない

- AIは補助機能として扱い、判断の主責務はアプリ側ロジックに置く
